# What-Ive-Fixed-On-My-Third-SQL-SSMS-Project.

This project demonstrates practical SQL data cleaning and analysis techniques using a retail shopping dataset.

It focuses on improving data quality, preventing errors, and generating meaningful business insights from customer purchase data.

🔧 Data Cleaning Techniques
Removing unwanted spaces using TRIM()
Handling missing values with ISNULL()
Preventing divide-by-zero errors using NULLIF()
Cleaning messy decimals using ROUND()
Rounding values with CEILING()
Detecting positive/negative values using SIGN()

📊 Data Analysis & Insights
Finding first purchase (MIN())
Identifying highest sale (MAX())
Calculating total revenue (SUM())
Ranking customers by spending (ORDER BY)
Retrieving top customers (TOP)
Filtering locations using LIKE

🎯 Purpose

The goal of this project is to demonstrate:
Strong understanding of SQL functions
Data validation and cleaning techniques
Business-focused data analysis
Writing readable and well-structured SQL queries
